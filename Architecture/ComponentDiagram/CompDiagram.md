# Диаграмма компонентов
![roguelike_component_diagram drawio](https://user-images.githubusercontent.com/54818274/199272943-2b75c9e2-f69f-46af-961c-7667cc9ade82.png)
Компоненты:

1. User Interactor - обеспечивает взамодействие с пользователем, то есть обрабатывает нажатия клавиш и движения мышки, а также управляет выводом на экран.
2. Game Model - отвечает за внутреннюю логику игры
3. Renderer - отвечает за вывод картинки на экран
